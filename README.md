📚 Library Management System (Python CLI)

A simple Library Management System built using Python. This project allows users to manage books efficiently using a command-line interface (CLI). It helps in performing basic library operations like adding books, issuing books, returning books, and viewing available books.

🚀 Features

Add new books to library
View all books with status (available/issued)
Issue books to users
Return issued books
Simple menu-driven CLI system

🛠️ Technologies Used

Python 3
Basic concepts: functions, loops, dictionaries, conditionals

▶️ How to Run

Clone this repository
Run the Python file:
python library.py
